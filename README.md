# C2050 Website

Official website for C2050 — a carbon credit ecosystem platform focused on jurisdictional REDD+ and nature-based solutions.

Built with [Next.js](https://nextjs.org), TypeScript, Tailwind CSS, and Three.js.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS
- **3D:** Three.js / React Three Fiber
- **Language:** TypeScript

## Project Structure

```
src/
  app/          # Pages (App Router)
  components/   # UI, layout, three.js, page-specific components
  content/      # Static content / data files
public/
  brand/        # Logos and brand assets
  icons/        # Service/feature icons
  images/       # Page images
  partners/     # Partner logos
  team/         # Team profile photos
scripts/        # Dev/debug utilities (CDP screenshot tools)
```
